Warnings

Usage

Warnings provide short and relevant information when users are about to do something that could be damaging or harmful.

Whenever possible, provide the user with an action that helps them address the warning. We can help the users with the content of the messages (text, icons, color) as well as the context (where on the screen and when in the user flow the messages appear).

Style

Icon

An exclamation mark enclosed inside a triangle indicates a warning. The icon is typically paired with warning text.

Color

Yellow is the dedicated color for warning states. Warning text should be highly legible, with noticeable contrast against its background color.
