Capitalization

Text in Firefox is nearly always title case (like most book or song titles) or sentence case (like most sentences in a blog post).

Title Case

For title case, you:

  • Always capitalize the first and last words of the text element.
  • Capitalize all other nouns, verbs, adverbs and pronouns.
  • Capitalize the second word in a hyphenated compound if it would normally be capitalized as a single word. For “Quick-Search Options,” capitalize both words. For “Add-on Choices,” only capitalize “Add.”
  • Don’t capitalize articles (“a,” “an,” “the”) unless they’re the first or last words.

Firefox for desktop

Use title case for menu labels, buttons, navigation, and section headers. (Also see Sentence Case and All Caps below.)

Firefox for iOS adheres to Apple’s Human Interface Guidelines.

Firefox for Android

In general, do not use title case.

Refer to the writing section of the Google-maintained Material Design Style Guide for complete Android capitalization rules.

Firefox for iOS

Use title case for menu labels, buttons, navigation, and section headers. (Also see Sentence Case and All Caps below.)

Firefox for iOS adheres to Apple’s Human Interface Guidelines.

Sentence Case

For sentence case, you:

  • Always capitalize the first word of the text element.
  • Capitalize any other words you would normally capitalize (e.g., Here is a poem Pablo wrote for Patricia about popular Polish pandas and the peculiar planet Pluto).

Firefox for desktop

Body copy is generally sentence case.

Firefox for Android

In general, use sentence case in Firefox for Android for all text elements except buttons.

Refer to the writing section of the Google-maintained Material Design Style Guide for complete Android capitalization rules.

Firefox for iOS

In general, use sentence case in Firefox for iOS for all text elements except buttons, navigation and section headers.

Firefox for iOS adheres to Apple’s Human Interface Guidelines.

All Caps

For sentence case, you capitalize every word in the text element.

Firefox for desktop

All caps in Firefox is generally not used. Hovewer, it can be used only in text heavy component, where the used of emphasis and of the de-emphasize is not enough.

Firefox for Android

Use all caps for buttons.

Firefox for iOS

Use all caps for sub-heads in app Settings (e.g., “ALLOW PODCASTS TO ACCESS” in Podcast Settings).
